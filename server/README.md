# Schedulify Server

- This is an Express + TypeScript project.

## Getting Started

### Installation

1. Install dependencies

```bash
npm install
```

2. Start the development server:

```bash
npm start
```

3. Build for production:

```bash
 npm build
```

4. Run API tests:

```bash
npm test
```

5. Lint for errors:

```bash
npm lint
```
